Installation Mode of Intelligent Power Braking System

  1. Set the vehicle power supply to OK.

  2. Use VDS to set the intelligent power braking system to "Installation Mode".

    1. Connect VDS, click “Vehicle Diagnosis System”, and select “Passenger Vehicles”.

    2. Select the corresponding vehicle series.

    3. Select the corresponding vehicle model.

    4. Click “Scan”.

    5. Select “EHB Module”.

    6. Select “Calibration”.

    7. Select “installation mode calibration”.

  3. Set the vehicle power supply to OFF.
