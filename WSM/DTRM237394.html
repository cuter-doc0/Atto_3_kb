Calibration of Front Millimeter Wave Radar (Veoneer Optional)

Static Calibration

Caution

After replacing the faulty components of millimeter wave radar, configuration writing and external parameter calibration are required to the ensure the normal driving assistance. Use a VDS to the perform the calibration.

  1. Site layout and tooling positioning parameters.

    1. The distance from the front axle center to the radar calibration plate is D: 2042mm.

    2. The distance from the center of the radar calibration plate to the ground is H: 353.5mm.

    3. The level of equipment calibration plate is displayed to the middle position.

  2. Drive the vehicle to the designated calibration site.

  3. Use a VDS to connect the vehicle OBD interface. Refer to the VDS Instructions

    Reminder
    OBD interface is located at left lower corner of left lower guard plate of instrument panel.

  4. Set the vehicle power supply to OK, enter "Vehicle Diagnostic System", and select the vehicle model.

  5. Select “Passenger Vehicles”.

  6. Select "Corresponding Vehicle Series".

  7. Select "Scan ECU".

  8. After scanning, select "Front Millimeter Wave Radar - (Veoneer)".

  9. Select “Action test”.

  10. Select "Ground Coordinate System".

    1. Write ground coordinate system data.

    2. Input ground coordinate system data: 12000.

    3. Tap the execute button.

  11. Return to the main page of the module and select "Calibration".

  12. Select "Static Calibration".

  13. When the calibration is completed, the instrument fault light will go out automatically.

    Caution

    If the calibration fails, adjust the calibration equipment angle according to the VDS interface prompt and recalibrate it.
