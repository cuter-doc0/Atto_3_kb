System Function Principle

  1. The infotainment system includes 4G network, PAD unit, Bluetooth phone, smart voice recognition, vehicle-infotainment interaction, on-board radio, USB, SD card assembly, audio and video equipment interface, image system and other functions.

  2. Main components:

    1. Multimedia host and multimedia display screen

    2. Speaker

    3. External antenna (internal antenna) and feeder

    4. 4G/WIFI antenna

    5. EDR

    6. Panoramic view system

    7. Rear view camera system

    8. USB interface assembly

  3. USB interface assembly:

    It includes 2 USB interfaces, of which only 1 USB interface supports audio and video file reading, and both USB interfaces can be used for charging handheld devices.

  4. MP3/WMA file playback:

    1. Playable MP3 files

    2. Playable WMA files

  5. 4G/WIFI antenna:

    It supports 4G internet access and Wi-Fi connection to phones, among other functions.

  6. Radio:

    Radio band FM:87.5MHz ~ 108.0MHz

  7. The Bluetooth key system of mobile phone supports the following functions:

    1. Mobile phone Bluetooth key unlock, switch on/off air conditioner and other vehicle control functions.

    2. Approach-to-unlock function.

    3. Mobile phone remote control driving function.
