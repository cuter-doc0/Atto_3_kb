System Function Principle

The intelligent power brake system includes the brake assist and brake adjustment modules, which replaces the traditional EVP+ESC modules, and the weight of the whole system is greatly reduced. Decoupling brake system is applied to solve the braking feedback problem of new energy vehicles. It mainly consists of the following parts:

  1. Hydraulic adjustment module of intelligent power brake system with electronic control unit

  2. Wheel speed sensor

  3. Steering wheel angle sensor (this module is not available if the steering system sends an angle signal)

  4. External inertial sensor

  5. Communication between modules of CAN network

Operating principle of intelligent power brake system

  1. Identifying the driver's intention: ECU can recognize the driver's intention through the position of steering wheel, wheel speed, accelerator pedal position and brake pressure.

  2. Identifying vehicle state: ECU calculates the control quantity required to maintain stability, and the hydraulic module quickly brakes individual wheel as required.

  3. Assisting steering through braking intervention: ECU calculates the control amount required to maintain stability, and the hydraulic module quickly brakes individual wheel as required.

Wheel speed sensor

The rotation speed of the wheel is an important input variable of the control system. The wheel speed sensor detects the rotation speed of the wheel and transmits the electrical signal to the control unit. The speed signal is used to calculate the degree of skid.

Steering wheel angle sensor

The intelligent power brake system selectively applies braking on each wheel to keep the vehicle in the desired direction. The steering angle is used as an important input to determine the desired direction. The angle signal is mainly transmitted in the following two forms:

  1. The steering wheel angle signal is sent by the steering system control module.

  2. Measure the angle value of the steering wheel and output it with standard CAN signal.

    Caution
    After the steering wheel angle sensor is installed on the vehicle, calibration is required. See Steering Angle Signal Calibration . This is to make sure that the middle position of the steering wheel is align with the 0° position of the steering wheel angle sensor (that is, the steering wheel angle is 0° when the vehicle is going straight).

Inertial sensor

The inertial sensor signal is used to calculate the actual movement of the vehicle. If the driver's target direction is very different from the actual movement of the vehicle, the inertial sensor will measure and analyze the angular rate, angular speed or inclination, and then ECU will send a control command to a wheels, and the intelligent power brake system will try to brake the vehicle.
