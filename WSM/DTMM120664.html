System Function Principle

  1. The vehicle is equipped with a smart key system (integrated into the left body control module). The driver can remotely unlock the door, power on and start the vehicle through the smart key.

  2. The intelligent access system supports electronic key, NFC key, cloud service and Bluetooth key, which are used through different communication modes and different key forms.

  3. The intelligent access system shall be compatible with AES and Hitag2 algorithms. If VCU/ECM is based on Hitag2 algorithm (bit 2.4 of 20A is 0, and 0 is sent by default), the anti-theft verification conducted by the intelligent access system and VCU/ECM shall be based on Hitag2 algorithm, which is consistent with the mass production state. If AES encryption algorithm is applied (bit 2.4 of 20A is 1, and 0 is sent by default), the newly defined anti-theft verification will be performed.

  4. The detection system is composed of 3 detection antenna assemblies and 1 high frequency receiver module (integrated into the left body control module) to detect the effective range inside and outside the vehicle.
