Yaw Rate Sensor Calibration

Calibration Procedure

Caution
After wheel alignment, all the steering wheel and the tires are in their center positions without subjecting any external forces.

  1. Set the vehicle power supply to OK.

  2. Use a VDS to connect the vehicle OBD interface. Refer to the VDS Instructions

    Reminder
    OBD interface is located at right lower corner of right lower guard plate of instrument panel.

  3. Select “Vehicle Diagnostic System”.

  4. Select “Passenger Vehicles”.

  5. Select “EV system”.

  6. Select the corresponding model and variant.

  7. Conduct the vehicle module scanning.

  8. Select "Electronic Body Stability System".

  9. Select “Wheel Air Discharge”

  10. Select "Yaw Rate Sensor Calibration".

  11. Conduct the calibration.

    1. After successful calibration, it will return to the previous interface.

  12. Select “Read DTC”.

  13. Select “Clear DTC”.

  14. VDS display: DTC cleared.

    1. Press any button to return to the previous interface.

  15. Select “Read DTC”.

  16. FDS display: No DTC.

    1. DTC cleared successfully, and the calibration process is completed.
