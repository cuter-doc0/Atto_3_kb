VIN Write Calibration of Drive Motor Control Module Assembly

VIN Write Calibration Procedure

Caution
On completion of the replacement of drive motor control module assembly, use VDS to carry out VIN write calibration.

  1. Use a VDS to connect the vehicle OBD interface. Refer to the VDS Instructions

    Reminder
    OBD interface is located at right lower corner of right lower guard plate of instrument panel.

  2. Set the vehicle power supply to OK, enter “Vehicle Diagnostic System”, and select the vehicle model.

  3. Conduct the vehicle module scanning.

  4. Enter the dynamic body control module, select “VIN Write and Read”.

  5. Select “VIN Write”, and select “Step 2 VIN Write”, and enter the corresponding VIN, then VDS displays “Write VIN Succeeded”.

  6. Select “VIN Read”, and select “Read Mode I: Read Data by Identifier SID 22h”, VDS displays “standard diagnostic mode switching succeeded”.

  7. Return to the previous page, select “VIN Read” and select “Read Mode II: Read Vehicle Information SID 09h”.

  8. Set the vehicle power supply to OFF then to OK, and the VIN write calibration of drive motor control module assembly is completed.
