Precautions

Caution

Important notes on smart key:

  • Never place the smart key in a hot area.

  • Never hit the smart key with hard objects or drop it.

  • Keep the smart key away from magnetic fields.

  • When the vehicle is locked and enters the anti-theft state, if the vehicle is not used, keep the smart key away, as the communication of the vehicle's automatic card search function will consume power.

Notes on possible smart key system failure:

  • The smart key battery is low.

  • There is a strong magnetic field or electric field (such as TV signal tower) near the detection system.

  • The smart key is shielded by metal objects.

  • The smart key is placed with a mobile phone.

  • The smart key system of another vehicle nearby is working.

  • If the smart key cannot be found even though it is within the detection range, put the smart key close to the magnetic card antenna.
