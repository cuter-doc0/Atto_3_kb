Anti-Theft Matching of Drive Motor Control Module Assembly

Anti-Theft Matching Procedure

Caution
Before removing the old drive motor control module assembly, use VDS to clear the password first, and then remove the drive motor control module assembly (if password clear cannot be realized with VDS due to drive motor control module assembly damage, skip Steps 1 and 2 below).

  1. Use a VDS to connect the vehicle OBD interface. Refer to the VDS Instructions

    Reminder
    OBD interface is located at right lower corner of right lower guard plate of instrument panel.

  2. Set the vehicle power supply to OK, enter “Vehicle Diagnostic System”, and select the vehicle model.

  3. Select “Special Diagnostic Function”, and select “Clear Motor Control Module Password”, the VDS displays “Password Cleared”, it indicates that the motor control module password has been cleared successfully.

    Caution
    Make sure that the key is placed near the detection antenna in the vehicle.

  4. Remove or install a new drive motor control module assembly, and repeat Step I.

  5. Select “Special Diagnostic Function”, and select “Motor Control Module Programming”, the VDS displays “Motor Control Module Programming Successfully”, it indicate that the motor control module programming has been completed .

  6. Set the vehicle power supply to OFF, and then to OK, the combination instrument will light up the OK indicator, it indicates the anti-theft matching of drive motor control module has been completed.
