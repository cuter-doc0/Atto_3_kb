Guide Pin Aftersales Package - Removal and Installation

Removal

Reminder
The removal and installation methods of the left and right guide pin aftersales package are the same. The left guide pin aftersales package is taken as an example.

  1. Remove the left rear wheel assembly. See Wheel Assembly - Removal and Installation 

  2. Remove 2 fixing bolts, take out, and put aside the left rear brake caliper sub-assembly aftersales package.

    Reminder
    Hang the rear brake caliper sub-assembly onto the rear shock absorber damper assembly with iron wires.

    • Tightening torque: 30±2N•m

  3. Remove the guided pin aftersales package from the left rear bracket assembly aftersales package.

    Reminder
    Clean the installation pin hole after removing the guide pin.

Install

Caution
Apply sufficient grease to the pin hole and guide pin before installation.

  1. The installation is performed in reverse order of removal.
