Cruise Configuration Calibration of Drive Motor Control Module Assembly

Cruise Configuration Calibration Procedure

Caution
On completion of the replacement of drive motor control module assembly, use VDS to carry out cruise configuration calibration.

  1. Use a VDS to connect the vehicle OBD interface. See the VDS Instructions

    Reminder
    OBD interface is located at right lower corner of right lower guard plate of instrument panel.

  2. Set the vehicle power supply to OK, enter “Vehicle Diagnostic System”, and select the vehicle model.

  3. Conduct the vehicle module scanning.

  4. Go to “Power and Control Module TI Module”

  5. Select “action test”, select “Cruise Configuration Calibration”, and select the corresponding option according to the current vehicle configuration (select "Constant Speed Cruise" or "Adaptive Cruise"). Then, the VDS interface prompts "Succeeded", it indicates that the cruise configuration calibration of drive motor control module assembly has been completed.

  6. Set the vehicle power supply to OFF then to OK, and the cruise configuration calibration of drive motor control module assembly is completed.
