Intelligent Power Braking System Model Configuration Write and Sensor Calibration

Calibration

Caution
After the intelligent power braking system assembly is replaced, execute model configuration write and (VPC) valve calibration first (in case of calibration failure, wait for 10 minutes before a new attempt), then carry out ESC internal sensor calibration.

  1. Set the vehicle power supply to OK.

  2. Use VDS to calibrate the intelligent power braking system.

    1. Connect VDS, click “Vehicle Diagnosis System”.

    2. Select “Passenger Car”.

    3. Select the corresponding vehicle series.

    4. Select the corresponding vehicle model.

    5. Click “Scan”.

    6. Select “EHB Module”.

    7. Select “Calibration”.

    8. Perform vehicle configuration writing.

    9. Select “End Line Calibration”.

    10. Select “Sensor Calibration”.

    11. Select “(VPC) valve calibration”

    12. Perform Internal Sensor Calibration

  3. Clear DTCs.

  4. Set the vehicle power supply to OFF, and then set it to OK to check whether the instrument gets a brake system alarm.
