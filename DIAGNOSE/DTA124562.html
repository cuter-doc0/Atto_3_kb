How to Handle Historical Fault

Although the fault is not shown in the current state, it did occur. In this case, related faults may always exist. When there are historical faults, perform the following detection steps:

  1. Check whether the user has replaced the fuse or disconnected the connector.

  2. If the relevant fuse has been replaced or the connector has been disconnected, clear the DTC and check whether the DTC has been reset. If the DTC is not reset, complete the diagnosis.

  3. If the DTC is reset, the relevant diagnostic DTC procedures need to be performed. Check the wiring harness and connector. See "How to Handle Intermittent Faults".
