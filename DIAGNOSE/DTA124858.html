C050200 Short Circuit Between Signal Line and Ground Line of Left Front Wheel Speed Sensor

DTC Description

C050200 Short Circuit Between Signal Line and Ground Line of Left Front Wheel Speed Sensor
Symptom Some functions of intelligent power brake system failed.
Possible cause

  1. Wiring harness or connector fault.

  2. Left front wheel speed sensor fault.

  3. Intelligent power brake control module fault.
Set fault condition -
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of intelligent power brake control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connectors of left front wheel sensor.
  1. Set the START/STOP button to OFF.
  2. Disconnect the wiring harness connector of left front wheel sensor B01(A).
  3. Check whether the wiring harness connector of left front wheel sensor is normal.
    4. Maintain or replace the wiring harness.
3
Check the left front wheel speed sensor signal line for short to ground.

  1. Measure the resistance between the wiring harness of left front wheel speed sensor connector B01 (A)-1 and the ground.

    3. Replace the front left speed sensor.
4
Check the wiring harness and connector of intelligent power brake control module.
  1. Disconnect the wiring harness connector of intelligent power brake control module B03.
  2. Check whether the wiring harness connector of intelligent power brake control module is normal or not.
    3. Maintain or replace the wiring harness.
