B2343 Clock Running Fault

DTC Description

B2343 Clock Running Fault

Symptom

Some functions of instrument panel cluster failed.

Possible cause

  1. Fuse fault.

  2. Wiring harness or connector fault.

  3. Instrument panel cluster fault.

  4. Multi-media fault.

Set fault condition

Communication between instrument panel and multi-media interrupted

Fault triggering condition

When the instrument panel cluster fails to receive the multimedia message for a certain period of time, the DTC will be generated.

Diagnostic Steps

1
Check the DTC of instrument panel cluster.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Clear DTCs.
  4. Set the START/STOP button to OFF, and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the communication network.
  1. Use a VDS to execute the network test.
  2. Check whether the central control large screen host has passed the network detection.

3
Check the fuse of central control large screen host.
  1. Check whether the instrument fuse box fuse F2/08 (20 A) is normal.
    2. Replace the fuse.
4

  2. Disconnect the wiring harness connector of central control large screen host G07(A).
  3. Check whether the central control large screen host wiring harness connector is normal.
    4. Maintain or replace the wiring harness.
