U011000 Communication with Motor Control Unit Failed

DTC Description

U011000 Communication with Motor Control Unit Failed

Symptom

-

Possible cause

  1. Wiring harness or connector fault.

  2. Front drive motor control unit fault.

  3. Power battery packet fault.

Set fault condition

Communication loss with front drive motor control unit (FMCU).

Fault triggering condition

When the vehicle is powered on, and the communication between battery execution and sampling unit and motor control module fails, the DTC will be generated.

Diagnostic Steps

1
Check the communication network.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Use a VDS to execute the network test.
  4. Check whether the front drive motor control module passes the network detection.

  3. Disconnect the wiring harness connector of integrated intelligent front drive control module BK45 (B).
  4. Check whether the integrated intelligent front drive control module connector is normal or not.
    5. Maintain or replace the wiring harness.
