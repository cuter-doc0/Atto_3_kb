B2A2712 Sunlight Sensor Circuit Shorted to Power

DTC Description

B2A2712 Sunlight Sensor Circuit Shorted to Power
Symptom Some functions of air conditioning system failed.
Possible cause

  1. Wiring harness or wiring harness connector fault.

  2. Sunlight sensor fault
Set fault condition When the sunlight sensor line is shorted to other power supplies, the DTC will be generated in the continuous memory and as needed.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connector of sunlight sensor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of sunlight sensor GaG04.
  3. Check whether the wiring harness connector of sunlight sensor is normal.
    4. Maintain or replace the wiring harness.
3
Check whether the radar sensor output signal line shorted to power.

  1. Set the START/STOP button to ON
  2. Measure the voltage between the wiring harness connector of sunlight sensor GaG04-7 and the ground.
