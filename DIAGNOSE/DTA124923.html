C059100 Supply Current of Brake Booster Motor Too Low

DTC Description

C059100 Supply Current of Brake Booster Motor Too Low
Symptom Some functions of intelligent power brake system failed.
Possible cause

  1. Fuse burn-out.

  2. Wiring harness or connector fault.

  3. Internal fault of intelligent power brake control module.
Set fault condition Brake booster motor power current low.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of intelligent power brake control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the fuse of intelligent power brake control module.
  1. Check whether the fuses F1/30(60A) and F1/44(60A) in the front compartment fuse box are normal.
    2. Replace the fuse.
3
Check the wiring harness and connector of intelligent power brake control module.
  1. Set the START/STOP button to OFF.
  2. Disconnect the wiring harness connector of intelligent power brake control module B03.
  3. Check whether the wiring harness connector of intelligent power brake control module is normal or not.
    4. Maintain or replace the wiring harness.
4
Check the ground line of intelligent power brake control module for open circuit.

  1. Check the resistance of the wiring harness connector of intelligent power brake control module B03-1, B03-30 to ground.
