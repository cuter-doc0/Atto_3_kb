B110B07 PM2.5 Quick Tester Air Pump Fault

DTC Description

B110B07 PM2.5 Quick Tester Air Pump Fault
Symptom The PM2.5 quick tester fails.
Possible cause The PM2.5 quick tester fails.
Set fault condition The PM2.5 quick tester air pump fails.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of PM2.5 tester.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”. Replace the PM2.5 tester .
