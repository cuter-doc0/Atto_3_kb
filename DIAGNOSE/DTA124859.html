C003100 Left Front Wheel Speed Sensor Signal Fault

DTC Description

C003100 Left Front Wheel Speed Sensor Signal Fault
Symptom Some functions of intelligent power brake system failed.
Possible cause

  1. Left front wheel speed sensor fault.

  2. There is excessive dirt on the transmission shaft of the drive motor.

  3. The transmission shaft ring gear of drive motor is damaged.
Set fault condition -
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of intelligent power brake control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the transmission shaft ring gear of drive motor and the left front wheel speed sensor.
  1. Check whether there are excessive dirt on the drive motor transmission shaft ring gear and the left front wheel speed sensor .
    2. Clean the drive motor transmission shaft ring gear and left front wheel speed sensor.
3
Check the transmission shaft ring gear of drive motor.
  1. Check whether the drive motor transmission shaft ring gear is damaged.
    2. Replace the drive motor transmission shaft. Replace the front left speed sensor.
