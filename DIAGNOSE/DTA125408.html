B133E00 low voltage PTC relay 1 fault

DTC Description

B133E00 Low Voltage PTC Relay 1 Fault
Symptom The heating function of the air conditioner is limited.
Possible cause

  1. Wiring harness or wiring harness connector fault.

  2. PTC relay 1 fault.

  3. The right domain control module fails.
Set fault condition -
Fault triggering condition Set the START/STOP button to ON.

Terminal Definition

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the front compartment electrical box wiring harness connector.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of front compartment electrical box B1D.
  3. Check whether the front compartment electrical box wiring harness connector is normal.
    4. Maintain or replace the wiring harness.
3
Check PTC1 relay
  1. Measure the resistance between the wiring harness connector pins of the PTC1 relay.
