Diagnostic Steps

1
Check the communication network of body network.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Use a VDS to execute the network test.
  4. Check whether the body network passes the network detection.
    5. Check for “intermittent fault”.
2
Check the terminal resistance of body network.

  1. Set the START/STOP button to OFF
  2. Measure the resistance between the wiring harness connector of OBD interface G03-3 and G03-11.

  3. Check whether the results are normal.
    4. Go to step 7.Go to step 7. 
3
Check the wiring harness connector of left domain control module.
  1. Disconnect the wiring harness connector of left domain control module G64(G).
  2. Check the wiring harness connector of left domain control module for corrosion, damage, pin withdrawing, etc.
  3. Check whether the results are normal.
    4. Maintain or replace the wiring harness.

    3. Replace the left domain control module .

Check the module on the body network.
  1. Set the START/STOP button to OFF
  2. Restore the module wiring harness connector on the body network.
  3. Disconnect one module wiring harness connector on the body network.
  4. Set the START/STOP button to ON
  5. Use a VDS to execute the network test.
  6. Check whether the body network passes the network detection.
    7. Replace the corresponding module. Check other modules according to the above steps.
