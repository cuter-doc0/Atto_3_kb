C117009 EPB Switch Fault (independent)

DTC Description

C117009 EPB Switch Fault
Symptom The parking brake fails.
Possible cause

  1. Wiring harness or connector fault.

  2. EPB switch fault

  3. The EPB control module fails.
Set fault condition EPB switch fault counter reading record.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of EPB control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connector of EPB switch.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of EPB switch G16.
  3. Check whether the wiring harness connector of EPB switch is normal.
    4. Maintain or replace the wiring harness.
3
Check the EPB switch
  1. Measure the resistance between the wiring harness connector pins of the EPB switch.
