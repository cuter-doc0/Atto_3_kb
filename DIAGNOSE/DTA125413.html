U02087 Communication with Humidity Sensor Lost

DTC Description

U02087 Communication with Humidity Sensor Lost
Symptom Some functions of the vehicle failed.
Possible cause

  1. Fuse burn-out.

  2. Wiring harness or wiring harness connector fault.

  3. LIN communication failure

  4. Humidity sensor fault
Set fault condition If the right domain control module fails to receive any message from the humidity sensor within a certain time period, the DTC will be generated.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Set the START/STOP button to OFF and wait for a few seconds.
  4. Set the START/STOP button to ON again, and read the DTC.
  5. Check whether the same DTC is displayed.
    6. Check for “intermittent fault”.
2
Check the humidity sensor fuse.
  1. Check whether the fuse in the front compartment fuse box F1/12 (7.5A) is normal.
    2. Replace the fuse.
3
Check the wiring harness connector of humidity sensor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of humidity sensor P17.
  3. Check whether the wiring harness connector of humidity sensor is normal or not.
    4. Maintain or replace the wiring harness.
4
Check the power of humidity sensor.
