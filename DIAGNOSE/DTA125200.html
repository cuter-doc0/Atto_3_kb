Diagnostic Steps

1
Check the communication network of PEPS network.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Use a VDS to execute the network test.
  4. Check whether the PEPS network has passed the network test.
    5. Check for “intermittent fault”.
2
Check the terminal resistor of PEPS network.

  1. Set the START/STOP button to OFF
  2. Measure the resistance between the wiring harness connector of OBD interface G03-1 and G03-2.

5
Check the wiring harness connector of right domain control module.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of right domain control module KG86(B).
  3. Check the wiring harness connector of right domain control module for normal function.
    4. Maintain or replace the wiring harness.

Check the modules on PEPS network.
  1. Set the START/STOP button to OFF
  2. Restore the wiring harness connectors of modules on the PEPS network.
  3. Disconnect one module wiring harness connector on the PEPS network.
  4. Set the START/STOP button to ON
  5. Use a VDS to execute the network test.
  6. Check whether the PEPS network has passed the network test.
    7. Replace the corresponding module. Check other modules according to the above steps.
