U043204 Yaw Rate Sensor Signal Communication Timeout

DTC Description

U043204 Yaw Rate Sensor Signal Communication Timeout
Symptom Some functions of intelligent power brake system failed.
Possible cause

  1. Fuse burn-out.

  2. Wiring harness or connector fault.

  3. Yaw rate sensor fault.

  4. Intelligent power brake control module fault.
Set fault condition Yaw rate sensor signal communication timeout.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of intelligent power brake control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the yaw rate sensor fuse.
  1. Check whether the front compartment fuse box fuse F1/42 (5 A) is normal.
    2. Replace the fuse.
3
Check the wiring harness and connector of yaw rate sensor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of yaw rate sensor GB76.
  3. Check whether the wiring harness connector of yaw rate sensor is normal.
    4. Maintain or replace the wiring harness.
