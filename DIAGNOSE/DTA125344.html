B11BE19 LIN2 Ambient Light Overloaded

DTC Description

B11BE19 LIN2 Ambient Light Overloaded

Symptom

Ambient light fails to be turned on or adjusting color

Possible cause

  1. Wiring harness or connector fault

  2. Ambient light holder fault

  3. Left domain control module fault

Fault setting conditions

Drive current ≥1A is detected continuously for 3s (current value matches)

Fault triggering condition

  1. The voltage of control module is between 9-16V

  2. LIN2 ambient light power pin fails to supply power

Diagnostic Steps

1
Check the DTC of left domain control module
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Use a VDS to record and then clear the DTC.
  4. Set the START/STOP button to OFF, and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check whether the left front door panel ambient light/left front door snap handle ambient light is replaced
  1. Check whether the left front door panel ambient light/left front door snap handle ambient light is modified or replaced.
  2. Check whether the results are normal.
    3. Replace them with the original parts.
3
Check whether the LIN wiring harness of left front door panel ambient light/left front door snap handle ambient light is connected to other equipment
  1. Check whether the LIN wiring harness of left front door panel ambient light/left front door snap handle ambient light is connected to other electrical equipment.
  2. Check whether the results are normal.
    3. Remove the external equipment and recover the vehicle.
4
Check the wiring harness connector of left front door panel ambient light/left front door snap handle ambient light
  1. Set the START/STOP button to OFF.
  2. Disconnect the wiring harness connector of left front door panel ambient light/left front door snap handle ambient light T13.
  3. Check the wiring harness connector of left front door panel ambient light/left front door snap handle ambient light for corrosion, damage, false connection, pin withdrawing, etc.
  4. Check whether the results are normal.
    5. Maintain or replace the damaged wiring harness or connector.
5
Check the wiring harness connector of left domain control module
  1. Disconnect the wiring harness connector of left domain control module G64(G).
  2. Check the wiring harness connector of left domain control module for corrosion, damage, false connection, pin withdrawing, etc.
  3. Check whether the results are normal.
    4. Maintain or replace the damaged wiring harness or connector.
6
Check the left front door panel ambient light/left front door snap handle ambient light
  1. Replace the left front door panel ambient light/left front door snap handle ambient light with a new one and restore the vehicle.
  2. Set the START/STOP button to ON, and conduct a test run to check its function.
  3. Check whether the results are normal.
    4. Replace the left front door panel ambient light/left front door snap handle ambient light. Replace the left domain control module .
