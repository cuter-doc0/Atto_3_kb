U1101 Communication Between Instrument and Combination Switch Interrupted

DTC Description

U1101 Communication Between Instrument and Combination Switch Interrupted

Symptom

-

Possible cause

  1. Fuse fault.

  2. Wiring harness or connector fault.

  3. Multi-function display fault.

  4. Multi-function switch fault

Set fault condition

Communication between instrument and combination switch interrupted

Fault triggering condition

DTC is generated when the instrument panel cluster does not receive a combination switch message for a certain period of time.

Diagnostic Steps

1
Check the communication network.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Use a VDS to execute the network test.
  4. Check whether the combination switch passes the network detection.
    5. Go to step 10.Go to step 10. 
2
Check fuses
  1. Check whether the front compartment fuse box fuse F1/37 (7.5 A) is normal.
  2. Check whether the instrument fuse box fuse F2/11 (7.5 A) is normal.
    3. Replace the fuse.
3
Check the wiring harness of combination switch and connector.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of combination switch G02.
  3. Check whether the wiring harness connector of combination switch G02 is normal.
    4. Maintain or replace the wiring harness.
4
Check the combination switch constant power.

  1. Set the START/STOP button to ON
  2. Measure the voltage between the wiring harness connector of combination switch G02-5 and the ground.
