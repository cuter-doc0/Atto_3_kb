Diagnostic Steps

1
Check the DTC of on-board charger.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Scan the ECU module.
  4. Go to on-board charger. Read and clear the DTC.
  5. Set the START/STOP button to OFF and wait for a few seconds.
  6. Set the START/STOP button to ON again.
  7. Go to on-board charger again and read the DTC.
  8. Check whether the same DTC is displayed.
    9. Check for “intermittent fault”.
2
Check the communication network.
  1. Use a VDS to execute the network test.
  2. Check whether the central control large screen host has passed the network detection.
    3. Go to step 9Go to step 9 
3
Check the fuse of the central control large screen host.
  1. Check whether the instrument fuse box fuse F2/08 (20 A) is normal.
    2. Replace the fuse
4
Check the central control large screen host wiring harness and connector
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of central control large screen host G07(A).
  3. Check whether the central control large screen host wiring harness connector is normal.
    4. Maintain or replace the wire wiring harness
5
Check the constant power supply of the central control large screen host.

  1. Measure the voltage between the central control large screen host wiring harness connector G07(A) -12 and the ground.
  2. Measure the voltage between the central control large screen host wiring harness connector G07(A) -13 and the ground.
  3. Measure the voltage between the central control large screen host wiring harness connector G07(A) -14 and the ground.
  4. Measure the voltage between the central control large screen host wiring harness connector G07(A) -15 and the ground.
  5. Measure the voltage between the central control large screen host wiring harness connector G07(A) -16 and the ground.

Check the central control large screen host CAN line.

  1. Measure the voltage between the central control large screen host wiring harness connector G07(A) -3 and the ground.
  2. Measure the voltage between the central control large screen host wiring harness connector G07(A) -19 and the ground.

9
Check the DTC of multi-media system.
  1. Go to multimedia system. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to multimedia system again and read the DTC.
  5. Check whether there is a DTC.
    6. Go to "Multimedia System" for diagnosis.
10
Check the DTC of right domain control module.
  1. Go to right domain control module. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to right domain control module again and read the DTC.
  5. Check whether other DTC exists.
    6. Go to “right domain control module" for diagnosis.
11
Check the DTC of other modules.
  1. Is the communication timeout fault with the central control large screen host read in other modules?
    2. Replace the central control large screen host. Replace the integrated intelligent front drive control module.
