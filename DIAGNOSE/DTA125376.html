B2A0813 Evaporator Outlet Refrigerant Temperature Sensor Open-circuited

DTC Description

B2A0813 Evaporator Outlet Refrigerant Temperature Sensor Open-circuited
Symptom Some functions of air conditioning system failed.
Possible cause

  1. Wiring harness or wiring harness connector fault.

  2. Evaporator outlet refrigerant temperature sensor fault.

  3. The right domain control module fails.
Set fault condition The sensor output voltage detected is higher than 4.95V.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connector of evaporator outlet refrigerant temperature sensor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of evaporator outlet refrigerant temperature sensor B13(A).
  3. Check whether the wiring harness connector of evaporator outlet refrigerant temperature sensor is normal.
    4. Maintain or replace the wiring harness.
3
Check the temperature sensor of the evaporator outlet refrigerant.
  1. Measure the resistance between the wiring harness connector pins of evaporator outlet refrigerant temperature sensor.
    Connector Condition Lower limit (kΩ) Upper limit (kΩ)
    (+) (-)

  2. Check whether the results are normal.
    3. Replace the temperature sensor of the evaporator outlet refrigerant.
4
Check the wiring harness connector of right domain control module.
  1. Disconnect the wiring harness connector of right domain control module BG86(B).
  2. Check whether the wiring harness connector of right domain control module is normal.
    3. Maintain or replace the wiring harness.
5
Check the signal line of the temperature sensor of the evaporator outlet refrigerant for open circuit.

  1. Measure the resistance between the wiring harness connector of evaporator outlet refrigerant temperature sensor B13(A)-3 and the wiring harness connector of right domain control module BG86(B)-52.
