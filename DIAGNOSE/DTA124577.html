U029787 Communication with On-board Charger Failed

DTC Description

U029787 Communication with On-board Charger Failed

Symptom

Do Not Charge.

Possible cause

  1. Fuse fault.

  2. Wiring harness or connector fault.

  3. Charger breakdown

  4. Power battery packet fault.

Set fault condition

Communication loss with on-board charger.

Fault triggering condition

When AC charging / AC VTOL discharging, the communication with the on-board charger is lost, the DTC will be generated.

Diagnostic Steps

1
Check the on-board charger communication network.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Use a VDS to execute the network test.
  4. Check whether the on-board charger passes the network detection.

3
Check the wiring harness and connector of integrated intelligent front drive control module.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of integrated intelligent front drive control module BK45 (A).
  3. Disconnect the wiring harness connector of integrated intelligent front drive control module BK45 (B).
  4. Check whether the integrated intelligent front drive control module connector is normal or not.
    5. Maintain or replace the wire wiring harness
