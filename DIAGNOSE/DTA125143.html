U014087 Communication with BCM Lost

DTC Description

U014087 BCM Communication Timeout
Symptom Fail to Charge.
Possible cause

  1. The left domain control module fails.

  2. The CAN wiring harness fault.

  3. Internal fault of wireless charging module.
Set fault condition BCM Message is not received within the time specified.
Fault triggering condition When the START/STOP button is set to ON and the system detects that BCM Message is not received within the specified time, the DTC will be generated.
