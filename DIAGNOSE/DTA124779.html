P1BB200 General Overtemperature Warning for Front Drive Motor

DTC Description

P1BB200 General Overtemperature Warning for Front Drive Motor

Symptom

The instrument reports that the drive function is limited.

Possible cause

  1. Cooling system faults

  2. Motor control module fault.

  3. Drive motor assembly fault.

Set fault condition

The motor temperature exceeds the specified threshold.

Fault triggering condition

When the vehicle is powered on and the motor temperature exceeds the specified threshold, the DTC will be generated.

Diagnostic Steps

1
Check the cooling system;
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Scan the ECU module.
  4. Go to power domain control module. Read and clear the DTC.
  5. Set the START/STOP button to OFF and wait for a few seconds.
  6. Set the START/STOP button to ON again.
  7. Go to power domain control module again and read the DTC.
  8. Check whether there is a DTC.
    9. Go to "Power Domain Control Module” for diagnosis.
2
Check the DTC of front drive motor control unit.
  1. Go to front drive motor control module. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to front drive motor control module again and read the DTC.
  5. Check whether the same DTC is displayed.
    6. The system is normal.
3
Check the motor winding temperature sensor.
  1. Set the START/STOP button to OFF.
  2. Remove the end cover at the connection between the integrated intelligent front drive control module and the drive motor copper bar.
  3. Disconnect the rotary transformer connector.
  4. Measure the resistance between terminals 1 and 6 of the rotary transformer connector.
    Connector Temperature (°C) Standard resistance (kΩ)
    (+) (-)
    1 6 30 2280
    -20 1190
    -10 646.9
    0 364.9
    10 212.5
    20 127.7
    30 78.88
    40 50.04
  5. Check whether the results are normal.
    6. Replace the drive motor assembly. .
4
Replace the integrated intelligent front drive control module and check DTC.
  1. Replace the integrated intelligent front drive control module.
  2. Connect the VDS to the data link connector.
  3. Set the START/STOP button to ON
  4. Scan the ECU module.
  5. Go to front drive motor control module. Read and clear the DTC.
  6. Drive the vehicle to an open field and drive it normally for about 10 min.
  7. Set the START/STOP button to OFF and wait for a few seconds.
  8. Set the START/STOP button to ON again.
  9. Go to front drive motor control module again and read the DTC.
  10. Check whether the same DTC is displayed.
    11. The system is normal. Replace the drive motor assembly. .
