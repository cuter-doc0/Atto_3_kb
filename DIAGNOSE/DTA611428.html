B134887 Communication with LIN throttle valve lost

DTC Description

B134887 Communication with LIN throttle valve lost

Symptom

Throttle valve failed.

Possible cause

  1. Wiring harness or connector fault.

  2. Fuse fault.

  3. Throttle valve fault.

  4. The right domain control module fault.

Fault setting conditions

—

Fault triggering condition

Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Scan the ECU module.
  4. Go to right domain control module. Read and clear the DTC.
  5. Set the START/STOP button to OFF and wait for a few seconds.
  6. Set the START/STOP button to ON again.
  7. Go to right domain control module again and read the DTC.
  8. Check whether the same DTC is displayed?
    9. Check for “intermittent fault”.
2
Check the instrument fuse box fuse.
  1. Check whether the instrument fuse box fuse F2/03 (7.5 A) is normal.
    2. Replace the fuse with a new one of the same specification.
3
Check the harness and connector of throttle valve.
  1. Set the START/STOP button to OFF
  2. Disconnect the throttle valve harness connector B85.
  3. Check whether the wiring harness connector of throttle valve is normal?
    4. Maintain or replace the wiring harness.
4
Check throttle valve power.

  1. Set the START/STOP button to ON.
  2. Measure the voltage between the wiring harness connector B85-2 of throttle valve and the ground.
