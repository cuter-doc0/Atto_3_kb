C003800 Supply Voltage of Left Rear Wheel Speed Sensor Low

DTC Description

C003800 Supply Voltage of Left Rear Wheel Speed Sensor Low
Symptom Some functions of intelligent power brake system failed.
Possible cause

  1. Wiring harness or connector fault.

  2. Left rear wheel speed sensor fault.

  3. Intelligent power brake control module fault.
Set fault condition -
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of intelligent power brake control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness and connector of left rear wheel sensor.
  1. Set the START/STOP button to OFF.
  2. Disconnect the wiring harness connector of left rear wheel sensor K08(A).
  3. Check whether the wiring harness connector of left rear wheel sensor is normal.
    4. Maintain or replace the wiring harness.
3
Check the power line of left rear wheel speed sensor for short to ground.

  1. Measure the resistance between the wiring harness connector of left rear wheel speed sensor K08(A)-2 and the ground.

5

  1. Set the START/STOP button to OFF.
  2. Disconnect the wiring harness connector of intelligent power brake control module B03.
  3. Check whether the wiring harness connector of intelligent power brake control module is normal or not.
    4. Maintain or replace the wiring harness.
6
Check the power line of left rear wheel speed sensor for open circuit.
