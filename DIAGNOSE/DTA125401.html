B2A3314 Front Blower Adjustment Signal Harness Shorted to Ground or Open-circuited

DTC Description

B2A3314 Front Blower Adjustment Signal Harness Shorted to Ground or Open-circuited
Symptom Functions of air volume adjustment of air conditioning system fail.
Possible cause

  1. Wiring harness or wiring harness connector fault.

  2. The right domain control module fails.

  3. Blower speed regulating resistor fault.
Set fault condition This DTC is generated when the right domain control module detects that the target adjustment signal voltage set by the blower does not match the actual feedback voltage.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connector of blower speed regulating resistors.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector BG24 of the blower speed regulating resistors.
  3. Check whether the wiring harness connector of blower speed regulating resistors is normal.
    4. Maintain or replace the wiring harness.
3
Check the speed control signal of front blower module short to ground.

  1. Measure the resistance between the wiring harness connector of front blower module BG24-3 and the ground.

4

  1. Disconnect the wiring harness connector of right domain control module BG86(B).
  2. Check whether the wiring harness connector of right domain control module is normal.
    3. Maintain or replace the wiring harness.
5
Check the front blower module speed regulation control signal for open circuit.
