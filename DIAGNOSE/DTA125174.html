B1B 5114 Signal Line of Left Rear Corner Sensor Shorted to Ground or Open-circuited

DTC Description

B1B 5114 Signal Line of Left Rear Corner Sensor Shorted to Ground or Open-circuited
Symptom Some functions of park assist system failed.
Possible cause

  1. Wiring harness or wiring harness connector fault.

  2. Left rear radar sensor fault.

  3. The left domain control module fails.
Set fault condition -
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of left domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the left rear radar sensor.
  1. Replace it with the normal left rear radar sensor and restore the vehicle.
  2. Set the START/STOP button to ON and conduct a test run to check its function.
  3. Check whether the results are normal.
    4. Replace the left rear radar sensor .
3
Check the wiring harness connector of left rear radar sensor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of left rear radar sensor R01.
  3. Check whether the wiring harness connector of left rear radar sensor is normal.
    4. Maintain or replace the wiring harness.
4
Check the signal line of the left rear radar sensor for short to ground.

  1. Measure the resistance between the wiring harness connector of left rear radar sensor R01-2 and the ground.

5
Check the wiring harness connector of left domain control module.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of left domain control module KG64(D).
  3. Check whether the wiring harness connector of left domain control module is normal.
    4. Maintain or replace the wiring harness.
6
Check the signal line of left rear radar sensor for open circuit.
