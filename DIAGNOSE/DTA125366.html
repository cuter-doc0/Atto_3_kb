B2A2013 Interior Temperature Sensor Open-circuited

DTC Description

B2A2013 Interior Temperature Sensor Open-Circuited
Symptom Some functions of air conditioning system failed.
Possible cause

  1. Wiring harness or connector fault.

  2. The Interior temperature sensor fails.

  3. The right domain control module fails.
Set fault condition When the detected output voltage of sensor is higher than 4.95V.
Fault triggering condition Set the START/STOP button to ON

Circuit Diagram

Terminal Definition

Connector Terminal number Terminal Definition

Inside temperature sensor

1 Ground
2 Exterior temperature sensor signal

Right domain body control module

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connector of interior temperature sensor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of interior temperature sensor G05.
  3. Check the wiring harness connector of interior temperature sensor for normal function.
    4. Maintain or replace the wiring harness.
3
Check the interior temperature sensor.
  1. Measure the resistance between the wiring harness connector pins of interior temperature sensor.
    Connector Condition Lower limit (kΩ) Upper limit (kΩ)
    (+) (-)
    1 2 -25°C 126.4 134.7

    1 2 10°C 19.68 20.35
    1 2 20°C 12.37 12.67
    1 2 30°C 7.95 8.14
    1 2 50°C 3.51 3.66
  2. Check whether the results are normal.
    3. Replace the interior temperature sensor .
4
Check the wiring harness connector of right domain control module.
  1. Disconnect the wiring harness connector of right domain control module G86(G).
  2. Check whether the wiring harness connector of right domain control module is normal.
    3. Maintain or replace the wiring harness.
5
Check the signal line of interior temperature sensor for open circuit

  1. Measure the resistance between the wiring harness connector of interior temperature sensor G05-2 and the wiring harness connector of right domain control module G86(G)-26.
