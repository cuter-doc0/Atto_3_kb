P1BC800 IGBT Three-phase Temperature Check Fault Alarm for Front Drive Motor Control Module

DTC Description

P1BC800 IGBT Three-phase Temperature Check Fault Alarm for Front Drive Motor Control Module

Symptom

The instrument reports that the drive function is limited.

Possible cause

  1. Cooling system faults

  2. Motor control module fault.

  3. Drive motor assembly fault.

Set fault condition

The IGBT temperature exceeds the specified threshold.

Fault triggering condition

When the vehicle is powered on, the temperature of IGBT exceeds the specified threshold, generating a DTC.

Diagnostic Steps

1
Check the cooling system;
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Scan the ECU module.
  4. Go to power domain control module. Read and clear the DTC.
  5. Set the START/STOP button to OFF and wait for a few seconds.
  6. Set the START/STOP button to ON again.
  7. Go to power domain control module again and read the DTC.
  8. Check whether there is a DTC.
    9. Go to "Power Domain Control Module” for diagnosis.
2
Check the data flow of front drive motor control unit.
  1. Place the vehicle at room temperature for two hours.
  2. Connect a VDS to the data link connector.
  3. Set the START/STOP button to ON
  4. Scan the ECU module.
  5. Go to on-board charger. Read and clear the DTC.
  6. Drive the vehicle to an open field and drive it normally for about 10 min.
  7. Set the START/STOP button to OFF and wait for a few seconds.
  8. Set the START/STOP button to ON again.
  9. Go to on-board charger again. And read the IGBT temperature data flow of the front drive motor control module.
  10. Check whether the temperature is normal.
    11. The system is normal. Replace the integrated intelligent front drive control module. .
