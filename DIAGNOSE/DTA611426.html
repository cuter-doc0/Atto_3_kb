U029600 Communication with Power Battery Management Module Lost

DTC Description

U029600 Communication with Power Battery Management Module Lost

Symptom

Some functions of power battery management module failed.

Possible cause

  1. Wiring harness or connector fault.

  2. Fuse fault.

  3. Battery management system fault.

  4. The left domain control module fails.

  3. Use a VDS to execute the network test.
  4. Check whether the integrated intelligent control module can pass the network test?

Check the wiring harness and connector of integrated intelligent control module.
