U015587 Communication with Instrument Panel Cluster Failed

DTC Description

U015587 Communication with Instrument Panel Cluster Failed
Symptom -
Possible cause

  1. Instrument panel cluster fault.

  2. The CAN wiring harness fault.

  3. Integrated intelligent control module fault.
Set fault condition Communication fault with instrument panel cluster.
Fault triggering condition DTC is generated if the system does not receive the instrument panel cluster message within a certain time.

Diagnostic Steps

1
Check the DTC of instrument panel cluster.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Scan the ECU module.
  4. Go to instrument panel cluster. Read and clear the DTC.
  5. Set the START/STOP button to OFF and wait for a few seconds.
  6. Set the START/STOP button to ON again.
  7. Go to instrument panel cluster again and read the DTC.
  8. Check whether the same DTC is displayed.
    9. Check for “intermittent fault”.
2
Check the communication network.
  1. Use a VDS to execute the network test.
  2. Check whether the instrument panel cluster passes the network detection.
    3. Go to step 9.Go to step 9. 
3
Check fuses
  1. Check whether the front compartment fuse box fuse F2/05 (5 A) is normal.
    2. Replace the fuse.
4
Check the instrument panel cluster wiring harness and connector.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of combination switch G01_1.
  3. Check whether the wiring harness connector of instrument panel cluster is normal.
    4. Maintain or replace the wiring harness.
5
Check the constant power of instrument panel cluster.

  1. Measure the voltage between the wiring harness connector of instrument panel cluster G01_1-1 and the ground.
    Connector Condition Voltage
    (+) (-)
    G01_1-4 Ground Throughout 11~14V
  2. Check whether the results are normal.
    3. Go to step 7.Go to step 7. 
6
Check the constant power of instrument panel cluster for open circuit.

  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of front compartment fuse box G82.
  3. Measure the resistance between the wiring harness connector of instrument panel cluster G01_1-4 and the wiring harness connector of instrument fuse box G82-6.
    Connector Condition Resistance
    (+) (-)
    G01_1-4 G82-6 Throughout Lower than 1Ω
  4. Check whether the results are normal.
    5. Maintain or replace the wiring harness. Replace the instrument fuse box, and the diagnosis is finished.
7
Check the ground circuit of instrument panel cluster.

  1. Measure the resistance between the wiring harness connector of instrument panel cluster G01_1-8 and the ground.
    Connector Condition Resistance
    (+) (-)
    G01_1-8 Ground Throughout Lower than 1Ω
  2. Check whether the results are normal.
    3. Maintain or replace the wiring harness.
8
Check the instrument panel cluster CAN line.

  1. Set the START/STOP button to ON
  2. Measure the voltage between the wiring harness connector of instrument panel cluster G01_1-2 and the ground.
  3. Measure the voltage between the wiring harness connector of instrument panel cluster G01_1-3 and the ground.
    Connector Condition Voltage
    (+) (-)
    G01_1-2 Ground Throughout 1.5~2.5V
    G01_1-3 2.5~3.5V
  4. Check whether the results are normal.
    5. Go to "CAN" for diagnosis Replace the integrated intelligent front drive control module. .
9
Check the DTC of instrument panel cluster.
  1. Go to instrument panel cluster. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to instrument panel cluster again and read the DTC.
  5. Check whether there is a DTC.
    6. Go to “instrument panel cluster" for diagnosis.
10
Check the DTC of right domain control module.
  1. Go to right domain control module. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to right domain control module again and read the DTC.
  5. Check whether other DTC exists.
    6. Go to “Right Domain Control Module" for diagnosis.
11
Check the DTC of other modules.
  1. Is the communication timeout fault with the instrument panel cluster read in other modules?
    2. Replace the instrument panel cluster . Replace the integrated intelligent front drive control module. .
