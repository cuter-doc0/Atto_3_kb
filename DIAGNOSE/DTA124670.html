B17A300 SRS CAN Signal Abnormal

DTC Description

B17A300 SRS CAN Signal Abnormal

Symptom

-

Possible cause

  1. Fuse fault.

  2. Wiring harness or connector fault.

  3. The gateway (integrated in the right domain) fails.

  4. Vehicle control unit fault.

Set fault condition

Abnormal SRS CAN signal is detected.

Fault triggering condition

When the vehicle is powered on, and abnormal SRS CAN signal is detected, the DTC will be generated.

Diagnostic Steps

1
Check the communication network.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Use a VDS to execute the network test.
  4. Check whether the airbag passes the network detection.
    5. Go to step 8.Go to step 8. 
2
Check the fuse in the front compartment fuse box.
  1. Check whether the fuse F1/38 (10A) in the front compartment fuse box is normal or not.
    2. Replace the fuse.
3
Check the wiring harness and connector of airbag control module.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of airbag control module KG10_1.
  3. Check whether the airbag control module connector is normal.
    4. Maintain or replace the wiring harness.
4
Check the IG1 power of airbag control module.

  1. Set the START/STOP button to ON
  2. Measure the voltage between wiring harness connector of airbag control module KG10_1-21 and the ground.
    Connector Condition Voltage
    (+) (-)
    KG10_1-21 Ground Throughout 11~14V
  3. Check whether the results are normal.
    4. Go to step 6.Go to step 6. 
5
Check the power of airbag control module for open circuit.

  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of front compartment fuse box B1C.
  3. Check the resistance between the wiring harness connector of airbag control module KG10_1-21 and the wiring harness connector of front compartment fuse box B1C-27.
    Connector Condition Resistance
    (+) (-)
    KG10_1-21 B1C-27 Throughout Lower than 1Ω
  4. Check whether the results are normal.
    5. Maintain or replace the wiring harness. Replace the front compartment fuse box .
6
Check the airbag control module ground line.

  1. Set the START/STOP button to OFF
  2. Measure the resistance between wiring harness connector of airbag control module KG10_1-20 and the ground.
    Connector Condition Resistance
    (+) (-)
    KG10_1-20 Ground Throughout Lower than 1Ω
  3. Check whether the results are normal.
    4. Maintain or replace the wiring harness.
7
Check the CAN line of airbag control module.

  1. Set the START/STOP button to ON
  2. Measure the voltage between wiring harness connector of airbag control module KG10_1-18 and the ground.
  3. Measure the voltage between wiring harness connector of airbag control module KG10_1-19 and the ground.
    Connector Condition Voltage
    (+) (-)
    KG10_1-18 Ground Throughout 2.5~3.5V
    KG10_1-19 1.5~2.5V
  4. Check whether the results are normal.
    5. Go to "CAN" for diagnosis Replace the airbag control module .
8
Check the DTC of airbag system.
  1. Go to airbag. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to airbag again and read the DTC.
  5. Check whether there is a DTC.
    6. Go to “Supplemental Restraint System”.
9
Check the DTC of power domain control module.
  1. Go to power domain control module. Read and clear the DTC.
  2. Set the START/STOP button to OFF and wait for a few seconds.
  3. Set the START/STOP button to ON again.
  4. Go to power domain control module again and read the DTC.
  5. Check whether other DTC exists.
    6. The system is normal. Replace the integrated intelligent front drive control module. .
