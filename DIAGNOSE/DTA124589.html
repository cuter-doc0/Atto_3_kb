P1A0300 BIC2 Working Abnormal

DTC Description

P1A0300 BIC2 Working Abnormal

Symptom

The instrument displays “Power battery failure”.

Possible cause

Internal fault of power battery pack.

Set fault condition

BIC working abnormal

Fault triggering condition

When the vehicle is powered on, and system detects abnormal operation of the BIC, the DTC will be generated.

Diagnostic Steps

1
Check the DTC of battery execution and sampling unit.
  1. Connect a VDS to the data link connector.
  2. Set the START/STOP button to ON
  3. Scan the ECU module.
  4. Go to battery execution and sampling unit. Read and clear the DTC.
  5. Set the START/STOP button to OFF and wait for a few seconds.
  6. Set the START/STOP button to ON again.
  7. Go to battery execution and sampling unit DTC again and read the DTC.
  8. Check whether the same DTC is displayed.
    9. Check for “intermittent fault”. Replace the power battery pack .
