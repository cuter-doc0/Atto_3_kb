B24d800 Handle Input Power Undervoltage

DTC Description

B24d800 Handle Input Power Undervoltage
Symptom The combination switch fails.
Possible cause

  1. Battery fault.

  2. Fuse burn-out.

  3. The wiring harness or wiring harness plug fails.

  4. Charging system malfunction

  5. Combination switch.
Set fault condition The supply voltage is less than 9V for not less than 2s.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of combination switch.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. The fault is triggered by disconnecting the negative pole of the battery or by low voltage.
2
Check the charging system.
  1. Set the START/STOP button to OK.
  2. Check whether the fault warning light of instrument charging system is on.
    3. Diagnose “charging system”.
3
Test the battery status.
  1. Set the START/STOP button to OFF.
  2. Perform a battery condition test.
  3. Does the status of the battery pass the test?
    4. Replace the battery .
4
Check the combination switch fuse.
  1. Check whether the instrument fuse box fuse F2/11 (7.5 A) is normal.
    2. Replace the fuse.
5
Check the wiring harness connector of combination switch.
  1. Disconnect the wiring harness connector of combination switch G02.
  2. Check whether the wiring harness connector of combination switch is normal.
    3. Maintain or replace the wiring harness.
6
Check the constant power of combination switch.

  1. Set the START/STOP button to ON.
