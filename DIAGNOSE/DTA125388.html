B2A2A14 Mode Motor Shorted to Ground or Open-circuited

DTC Description

B2A2A14 Mode Motor Shorted to Ground or Open-circuited
Symptom The mode function of air conditioner fails.
Possible cause

  1. Wiring harness or wiring harness connector fault.

  2. Mode motor fault

  3. The right domain control module fails.
Set fault condition When the right domain control module fails to detect the reference voltage of mode motor, the DTC will be generated in the continuous memory and as needed.
Fault triggering condition Set the START/STOP button to ON.

Diagnostic Steps

1
Check the DTC of right domain control module.
  1. Connect the VDS to the data link connector.
  2. Set the START/STOP button to ON.
  3. Clear DTCs.
  4. Set the START/STOP button to OFF and wait for a few seconds.
  5. Set the START/STOP button to ON again, and read the DTC.
  6. Check whether the same DTC is displayed.
    7. Check for “intermittent fault”.
2
Check the wiring harness connector of mode motor.
  1. Set the START/STOP button to OFF
  2. Disconnect the wiring harness connector of mode motor G22(C).
  3. Check whether the wiring harness connector of mode motor is normal.
    4. Maintain or replace the wiring harness.
3
Check the mode motor.
  1. Measure the resistance between the pins of the wiring harness connector of mode motor.
